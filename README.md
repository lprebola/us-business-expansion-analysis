# U.S. Business Expansion Analysis

## Overview
A data analytics pipeline that ranks all 50 U.S. states by business attractiveness using 2024 federal economic data.

## Data Sources
 - BLS Local Area Unemployment Statistics (2024) 
 - U.S. Census Bureau ACS 1-Year Estimates (2024)
 - U.S. Census Bureau Population Estimates (2024)

## Methodology
The ranking of the states were based off of three metrics of differing weights
 - Median household income = 40%
 - Unemployment Rate = 40%
 - Population size = 20%

## Results
Top 5 States for business expansion:
1. California
2. Texas
3. Florida
4. New York
5. Maryland

## Tools Used
 - Python (pandas) = Used for data cleaning and scoring
 - Tableau Public = Used to construct interactive dashboard
 - Github = Used for version control
